# 공정무역과 블록체인
> 현은채 김현경 최은수

## 현재 문제점
- 빈민국에서 재배하거나 제작한 물건을 부유한 대기업이 불공정무역을 이용해서 빈민국 사람들이 정당한 대가를 받지 못하고 있다.

## 해결 방안
- 유통과정과 그 과정에서의 비용등을 생산자에게 모두 개방해 합의된 공정한 가격을 생산자에게 지불한다.
